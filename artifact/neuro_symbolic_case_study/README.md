# Neuro-Symbolic Disaster Loop Case-Study Bundle

This directory contains the small public artifact bundle used by the paper:

- `danger_atlas.json`: the MPRD case-study danger atlas.
- `neuro_symbolic_disaster_loop_latest.json`: compact stable JSON receipt.
- `neuro_symbolic_disaster_loop_latest.md`: Markdown rendering of the receipt.

The receipt is sufficient to reproduce the paper's reported case-study counts,
gate status, and stable receipt hash. Full regeneration of the receipt requires
the complete local assurance bundle referenced by the checker.
